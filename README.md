# 개요
> 사용자의 표정 이미지에서 감정을 인식해 그날 하루의 기분을 기록하는 플랫폼이다. 따로 자신의 기분을 설명할 필요 없이 자신의 셀피로만 일상의 감정을 기록할 수 있다.
